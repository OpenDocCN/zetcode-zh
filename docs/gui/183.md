## Winforms

Windows Forms 是图形用户界面应用编程接口（API），包含在 Microsoft .NET Framework 中。 截至 2008 年 5 月 13 日，Mono 的`System.Windows.Forms` 2.0 已完成 API。 简而言之，Winforms 是一个用于创建 GUI 应用的库。
