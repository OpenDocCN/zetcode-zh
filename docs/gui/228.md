## GTK# 

GTK# 是针对 C# 编程语言的 GTK+ 的包装。 该库有助于使用 Mono 或任何其他兼容的 CLR 构建图形 GNOME 应用。 使用 Gtk# 构建的应用将在许多平台上运行，包括 Linux，Microsoft Windows 和 Mac OSX。GTK# 是 Mono 计划的一部分。
