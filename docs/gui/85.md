## wxWidgets

wxWidgets 是用于创建 C++ 应用的 GUI（图形用户界面）工具箱。 它是一个开放源代码，成熟且跨平台的工具箱。 wxWidgets 应用可在所有主要的 OS 平台，Windows，Unix 和 Mac OS 上运行。
