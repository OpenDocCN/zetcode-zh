## PHP

PHP 是一种专为 Web 开发而设计的脚本语言。 它用于产生动态网页。 当前，PHP 是使用最广泛的编程语言之一。 它的大部分语法是从 C，Java 和 Perl 借来的，它们具有几个特定于 PHP 的独特功能。 PHP 可以嵌入 HTML 代码中，并且通常在 Web 服务器上运行。
