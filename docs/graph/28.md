## Cario

Cario 是用于创建 2D 矢量图形的库。 它是用 C 编程语言编写的。 还有其他计算机语言的绑定。 Python，Perl，C++ ，C# 或 Java。 Cario 是一个多平台库。 它适用于 Linux，BSD，OSX。
