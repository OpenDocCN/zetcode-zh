## 安卓系统

Android 是基于 Linux 的操作系统，专为智能手机和平板电脑等移动设备设计。 它还用于各种网络设备，智能电视系统，手表和家用电器。 Android 的应用是使用 Java 编程语言的自定义版本开发的。
