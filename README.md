## 下载

### Docker

```
docker pull apachecn0/zetcode-zh
docker run -tid -p <port>:80 apachecn0/zetcode-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g zetcode-zh
zetcode-zh <port>
# 访问 http://localhost:{port} 查看文档
```
